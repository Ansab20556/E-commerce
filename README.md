# E-commerce Project

## Overview
This is a complete E-commerce web application built with **Laravel 12**. The project includes a fully functional **Products CRUD system** and a **Contact Us form**. Users can view, add, edit, and delete products, as well as send messages through the contact form.

---

## Features

### Products CRUD
- Create, Read, Update, Delete products
- Product images upload
- Pagination for product listings
- Slug generation for SEO-friendly URLs

### Contact Form
- Users can submit inquiries via the "Contact Us" page
- Messages are stored in the database
- Basic form validation
